Color system

Four disciplined colors. No gradients, no noise.

Background #F7F6F3

Default canvas. Avoid pure white to keep visual fatigue low.

Primary text #111111

Body and label text. No secondary gray palette.

Accent #8B0000

Primary action and emphasis. Use sparingly.

Success Muted green

Confirmation only states. Never used as a primary action.

Warning Muted amber

Soft warnings and caution banners. No alarm tones.

Usage rules

  • Background remains off-white on every surface; no alternating stripes.
  • Deep red accent is reserved for primary actions and key highlights.
  • Success and warning colors appear only in inline feedback and banners.
  • No additional brand or chart palettes are introduced at this layer.

Typography & spacing

Calm hierarchy with deliberate whitespace.

HEADING (SERIF)

Confident, measured product narrative

BODY (SANS-SERIF)

KodNest text is set in a modern, system-aligned sans-serif at 16–18px with a line height between 1.6 and 1.8. Each paragraph is constrained to a comfortable reading width so dense content remains scannable.

Spacing scale

All vertical and horizontal rhythm uses a single scale. No arbitrary gaps, no magic numbers.

8px Fine padding, dense inline controls.
16px Default padding and line separation.
24px Section dividers and card gutters.
40px Major layout breathing room.
64px Page-level framing and empty states.
